# 🚀 Installing Python

## Step 1 — Download Python

Visit the official Python website:

👉 https://www.python.org/downloads/

Download the latest stable version for your operating system.

---

## Step 2 — Install Python

Run the downloaded installer.

### **Important**

Before clicking **Install Now**, make sure you enable:

```text
☑ Add Python to PATH
```

Then click:

```text
Install Now
```

Wait until the installation completes.

---

## Step 3 — Verify Installation

Open **Command Prompt**, **PowerShell**, or **Terminal** and run:

```bash
python --version
```

or

```bash
py --version
```

Expected output:

```text
Python 3.13.x
```

If you see the Python version, the installation was successful.

# ▶️ Running Python Files

Run any Python file using:

```bash
python filename.py
```

or

```bash
py filename.py
```

Example:

```bash
python "Basics/1. Variables and Data Types.py"
```

or

```bash
py "Basics/1. Variables and Data Types.py"
```
